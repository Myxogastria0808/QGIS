Whai is my creation

I developed a GIS map of the flood damage of July, 2008. I use QGIS and QGIS2Web. QGIS is one of the FOSS4G stands for Free Open-Source Software for GeoSpatial. QGIS2Web is a plug-in of QGIS to release the Internet.

The hazard map using QGIS

In July 2008, Japan experienced a major rainstorm disaster. Many people were affected by this disaster. Japan has experienced many flood disasters like this to data. To prepare for future floods, we must learn from the past disaster records. So, I have developed the hazard map in Osu City, Ehime Prefecture using some GIS data created by the Geospatial Information Authority of Japan. I realized to put an interactive hazard map on the website using QGIS2web! Let's see the dynamic hazard map which you have never seen yet!
